# Index

Einstiegspunkt für dieses Repository. Von hier aus ist alles erreichbar.

## Arbeitsweise

Neue Gedanken landen zuerst in der [Inbox](inbox.md) — ohne Sortierung, ohne Anspruch. Beim Verarbeiten wandern sie in den passenden Bereich oder werden gelöscht, wenn sie sich erledigt haben.

## Bereiche

| Bereich | Inhalt |
|---|---|
| [inbox.md](inbox.md) | Unsortierte Gedanken, die später verarbeitet werden |
| [personality/](personality/) | Wer ich bin: Identität, Werte, Stärken, Schwächen, Muster |
| [goals.md](goals.md) | Kurz- und langfristige Ziele |
| [relationship/](relationship/) | Meine Beziehung zu Angeli |
| [people/](people/) | Andere Menschen in meinem Leben, eine Datei pro Person |
| [decisions/](decisions/) | Wichtige Entscheidungen mit Kontext und Rückschau, eine Datei pro Entscheidung |
| [health/](health/) | Gesundheitliche Themen — körperlich und mental |
| [documents/](documents/) | Wichtige Unterlagen (Scans, PDFs), thematisch in Unterordnern |
| `archive/` | Überholte Inhalte (entsteht bei Bedarf, nichts wird einfach gelöscht) |

## Konventionen

- Inhalte auf Deutsch, Datei- und Ordnernamen auf Englisch, kleingeschrieben, mit Bindestrich.
- Lieber roh und ehrlich als perfekt formuliert. Lieber kurz festhalten als gar nicht.
- Überholtes wird nach `archive/` verschoben, nicht gelöscht.
